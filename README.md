# Bank Loan Risk Analytics

SQL-based banking analytics project that analyzes loan applications, funded amounts, repayments, loan quality, and default risk to help financial institutions make data-driven lending decisions.

---

## Project Overview

Banks process thousands of loan applications every month, but identifying good loans, bad loans, repayment trends, and risk factors is a major challenge.

This project analyzes loan data using MySQL to generate business insights that help banks:

✅ Identify high-risk borrowers  
✅ Track repayment performance  
✅ Monitor funded amounts  
✅ Measure loan quality  
✅ Improve lending decisions  

---

## Business Problem

Financial institutions often face challenges such as:

- Increasing loan defaults  
- Difficulty identifying reliable borrowers  
- Lack of visibility into monthly lending trends  
- Poor understanding of state-wise and customer-wise loan performance  

Without proper analytics, banks may approve risky loans and lose revenue.

---

## Solution

This project uses SQL analytics to solve these business problems by building KPIs and analytical reports.

---

## Loan Performance KPIs

- Total Loan Applications  
- Total Funded Amount  
- Total Amount Received  
- Average Interest Rate  
- Average DTI Ratio  

---

## Risk Analysis

- Good Loan Percentage  
- Bad Loan Percentage  
- Charged-Off Analysis  
- Loan Status Analysis  

---

## Business Insights

- Month-wise Loan Trends  
- State-wise Loan Distribution  
- Loan Purpose Analysis  
- Employment Length Analysis  
- Home Ownership Analysis  
- Term-based Loan Analysis  

---

## Key Business Questions Solved

1. How many loan applications were received?  
2. What is the total funded amount?  
3. What percentage of loans are good vs bad?  
4. Which states generate the highest loan volume?  
5. Which loan purposes carry higher risk?  
6. How does repayment vary across customer segments?
